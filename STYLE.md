# Стиль Драконоборца

## Принцип
Сухой список цифр + один нейтральный комментарий в конце каждого блока. Без образных глаголов («разорвал», «подсдулся», «молчит»). Директор читает за кофе — ему нужны цифры и короткая сводка, не развлекательный текст.

## Правила

1. **Каждая строка — одна цифра и её разбивка.** Без оценки, без комментария к строке.
   - ✅ `n8n-web-1 — 18 (Директ 18)`
   - ❌ `n8n-web-1 — 18, чистый Директ, локомотив`

2. **Один общий комментарий на блок** — в последней строке блока после тире, 1-2 предложения. Нейтрально-наблюдательно.
   - ✅ `— Главный объём держат RIS (на ai-google) и Директ (на prompt-engineer и n8n).`
   - ❌ `— Директ тащит всё на себе, остальные в жопе.`

3. **Не упоминай нулевые каналы/лончи.** Если канала нет во вчерашнем списке — его просто нет в отчёте. Никаких «молчит», «тишина», «нули».

4. **Сравнение с позавчера** только для детского (есть готовые числа в JSON как `channels_prev`). Формат `(+N)` или `(−N)` в конце строки.

5. **Схлопывай длинный хвост.** Если лончей много — первые ~10 с цифрами, остальные одной строкой: `Ещё 11 лончей по 1-4 рег.`

6. **Тон комментария** — фактический. Что видишь в данных, то и говоришь. Без драмы и без рекомендаций.

## Хорошо

```
🎪 ВЗРОСЛОЕ: 220 новых рег на вебинары
ai-google-web-1 — 57 (RIS 56, Timepad 1)
prompt-engineer-web-14 — 26 (Директ 23, Timepad 2)
n8n-web-1 — 18 (Директ 18)
...
Ещё 11 лончей по 1-4 рег.
— Главный объём держат RIS (на ai-google) и Директ (на prompt-engineer и n8n).
```

## Плохо

```
🎪 ВЗРОСЛОЕ: 220 новых рег!
ai-google-web-1 разорвал — 57 рег, из них 56 с RIS, их канал, гнём туда дальше!
n8n-web-1 — 18, чистый Директ как в учебнике
Flocktory молчит третий день
```

## Что НЕ писать
- Эмоциональные образы: разорвал, подсдулся, молчит, вытянул, проебал
- Рекомендации: «надо докрутить», «стоит добавить», «пора копать»
- Прилагательные-наполнители: классно, круто, неплохо, отлично
- Приветствия: «Доброе утро!», «Сегодня в Zerocoder»
- Объяснения причин, если не видишь их в данных
